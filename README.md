# 🦀 RUSTy Programming 🦀

Solutions to **RUST College Class** exercises as well as some small personal projects. The goal was to learn a new language and also make some interesing projects along the way.

## How to compile and boot

The project uses **rustc** to run rust files and **cargo** to create and manage rust projects.

### 📄 RUSTC - Compiling the programs

To compile a rust file pick a file that has a ``.rs`` extension and use the following command:
```sh
rustc file.rs
```
To run the compiled program use
```sh
rustc ./file
```

### 💾 CARGO - Building / Running projects

To build and compile a cargo project locate a directory containing cargo files and run the following command:
```sh
cargo build
```
The compiled project can be ran using this command:
```sh
cargo run
```
